# Distribution Manager

Handle release submission, status tracking, and platform management.

## When to Activate

- "Upload my track," "submit to DistroKid," "is my release live?"
- "Take down a track," "update metadata," "check release status"
- Artist needs help with distributor workflow

## Submission Checklist

Before submitting any release, verify:

### Audio
- [ ] Format: WAV or FLAC (16-bit/44.1kHz minimum, 24-bit preferred)
- [ ] Loudness: -14 LUFS (Spotify target), -16 LUFS (Apple Music)
- [ ] No clipping, no silence at start/end (unless intentional)
- [ ] Filename matches track title

### Artwork
- [ ] Dimensions: 3000x3000px (minimum 1400x1400)
- [ ] Format: JPEG or PNG, RGB color space
- [ ] No blurry/pixelated images
- [ ] No social media logos, URLs, or "available on Spotify" text
- [ ] No explicit content visible (unless marked explicit)

### Metadata
- [ ] Track title (no ALL CAPS unless stylistic)
- [ ] Artist name (consistent across releases)
- [ ] Featured artists (use "feat." format)
- [ ] Genre and subgenre
- [ ] Release date (Friday recommended)
- [ ] Copyright year and holder
- [ ] ISRC code (auto-generated by most distributors)
- [ ] UPC/EAN (auto-generated for new releases)
- [ ] Explicit content flag (if applicable)
- [ ] Lyrics (for Apple Music, Spotify lyrics display)
- [ ] Composer/songwriter credits
- [ ] Producer credits

## Platform Timeline

After distributor submission, typical availability:
- Spotify: 2-5 business days
- Apple Music: 1-3 business days
- YouTube Music: 2-5 business days
- Amazon Music: 2-5 business days
- Tidal: 3-7 business days
- Deezer: 2-5 business days
- TikTok/Instagram: 1-3 business days

**Always submit 3-4 weeks before release date** to ensure platform processing + editorial submission window.

## Common Issues

| Problem | Fix |
|---|---|
| Release not showing on Spotify | Check distributor dashboard for errors. Spotify indexing takes 24-48h after going live. |
| Wrong artist profile | Claim profile on Spotify for Artists. Contact distributor for profile merge. |
| Metadata rejection | Check for special characters, ALL CAPS violations, or missing credits. |
| Artwork rejection | Verify dimensions, no text overlays with platform names, no blurry images. |
| Duplicate ISRC | Each track needs a unique ISRC. Don't reuse across different releases. |
