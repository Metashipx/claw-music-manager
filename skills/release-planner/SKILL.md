# Release Planner

Create comprehensive release plans for singles, EPs, and albums.

## When to Activate

- Artist mentions a new release, single, EP, or album
- "Plan my release," "when should I drop this," "release strategy"
- A release date is set and no plan exists yet

## Release Plan Template

When creating a plan, follow this structure. Adjust timelines based on whether it's a single (4-6 weeks) or album (8-12 weeks).

### Single Release — 6-Week Plan

**Week 6 (T-6):** Foundation
- [ ] Finalize master audio (WAV, 16-bit/44.1kHz minimum)
- [ ] Commission cover artwork (3000x3000px JPEG, RGB)
- [ ] Write metadata: title, artist, genre, mood, language, ISRC
- [ ] Decide release date (Fridays globally — "New Music Friday" alignment)

**Week 5 (T-5):** Distribution
- [ ] Submit to distributor (DistroKid/TuneCore)
- [ ] Submit to Spotify editorial playlists (via Spotify for Artists)
- [ ] Set up pre-save campaign (feature.fm, linkfire, or similar)
- [ ] Create Apple Music pre-add

**Week 4 (T-4):** Press & Outreach
- [ ] Draft press release (use press-kit skill)
- [ ] Send to music blogs / press contacts
- [ ] Begin playlist curator outreach (use playlist-pitcher skill)
- [ ] Reach out to sync licensing contacts if applicable

**Week 3 (T-3):** Social Warm-Up
- [ ] First teaser post (snippet, behind-the-scenes, studio clip)
- [ ] Share pre-save link
- [ ] IG Story countdown sticker
- [ ] Fan engagement push (replies, DMs, comments)

**Week 2 (T-2):** Ramp Up
- [ ] Second teaser (different angle — lyrics, artwork reveal, collab tease)
- [ ] TikTok using snippet with trending format
- [ ] Email fan list (Klaviyo) with pre-save CTA
- [ ] Confirm all platform submissions are processed

**Week 1 (T-1):** Launch Week
- [ ] Daily social content
- [ ] Release day post: Spotify link, Apple Music, YouTube (if video)
- [ ] IG/TikTok live on release day (optional)
- [ ] Email blast: "IT'S OUT"
- [ ] Thank fans who pre-saved

**Post-Release (Weeks 1-4):**
- [ ] Track streaming numbers daily
- [ ] Monitor playlist adds
- [ ] Share milestones (1K streams, playlist add, etc.)
- [ ] Pitch for additional playlist adds based on momentum
- [ ] Run targeted social ads if budget allows

### Key Dates to Calculate

Given a release date, auto-calculate:
- Distributor submission: Release date - 4 weeks minimum
- Spotify editorial submission: Release date - 4 weeks
- Press outreach: Release date - 3 weeks
- First social teaser: Release date - 3 weeks
- Pre-save launch: Release date - 4 weeks
- Email campaign: Release date - 2 weeks, then release day

### Platform-Specific Notes

- **Spotify**: Submit editorial pitch via Spotify for Artists. Include: song story, mood, genre, instruments, similar artists. Tuesday submissions historically perform better.
- **Apple Music**: Pre-add available through distributor. No editorial pitch portal — relationships matter.
- **YouTube**: Upload music video or lyric video. Premiere feature for countdowns. Optimize title/description for search.
- **TikTok**: Don't push the full song link. Let the sound go viral organically. Create 2-3 short clips using the catchiest 15-second section.
